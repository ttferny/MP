How SPF Works
  1. Mail server extracts the envelope domain from Return-Path.
  2. It fetches the SPF record from DNS (TXT record).
  3. Each mechanism is tested in order until one matches.
  4. The matching qualifier decides the result (pass, fail, softfail, neutral).
  5. If nothing matches, the result is neutral.
